# 🚨 Emergency Response System

A complete emergency management system for citizens, dispatchers, and police officers with AI-powered classification.

## ✨ Features

- 📱 **Citizen Reporting** - Report emergencies with GPS location
- 🤖 **AI Classification** - Automatic emergency categorization
- 🗺️ **Live Map** - Real-time emergency visualization
- 🖥️ **Dispatcher Dashboard** - Manage all emergencies
- 👮 **Police Dashboard** - Handle assigned cases
- 📊 **Complaint Tracking** - Real-time status updates
- 🔔 **Notifications** - Status updates for citizens

## 🛠️ Tech Stack

### Frontend
- React 19 with Vite
- Tailwind CSS
- Leaflet for maps
- React Router for navigation
- Axios for API calls

### Backend
- Node.js with Express
- Firebase Firestore (Database)
- Firebase Auth (Authentication)
- Google Gemini AI (Classification)
